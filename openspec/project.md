# 项目规范：NoteCascade v2.3.0

## 概览
NoteCascade 是一款视觉化 MIDI 键盘练习应用，旨在通过类游戏的界面帮助用户掌握钢琴技能。它具有瀑布流式的音符显示、实时评分和成就系统。

## 核心功能
- **实时 MIDI 输入**：通过 Web MIDI API 连接物理 MIDI 键盘（支持 USB 和蓝牙）。
- **视觉反馈**：与所选歌曲同步的下落音符，支持完整下落动画。
- **评分系统**：Perfect（完美）、Good（良好）、Miss（错过）和 Wrong（错误）击键检测。
- **曲库**：内置不同难度等级的歌曲，支持 MIDI 文件上传，响应式布局。
- **成就系统**：练习里程碑的可解锁奖励。
- **音频反馈**：为演奏的音符提供高质量的合成器声音。
- **练习模式**：音符在未正确按下时停止下落，辅助练习。
- **自由演奏 (Free Play)**：无评分、无反馈的自由弹奏模式，已统一标识符为 `free-play`。
- **演示模式 (Demo)**：自动播放乐曲并显示按键提示。
- **多语言支持**：支持 11 种语言（中/英/日/韩/俄/德/法/西/葡/阿），已同步所有语言文件的 Key。
- **MIDI 映射**：支持物理按键到虚拟按键的自定义映射。
- **节拍器**：内置可调节 BPM 的节拍器。

## 技术栈
- **框架**：Next.js 15+ (App Router)
- **语言**：TypeScript
- **样式**：Tailwind CSS v4
- **动画**：Motion (motion/react)
- **音频**：Tone.js
- **状态管理**：Zustand
- **图标**：Lucide React
- **PWA**：支持离线访问和桌面安装 (@ducanh2912/next-pwa)

## UI/UX 特性
- **顶部工具栏**：集成节拍器、音量控制、成就入口和全屏切换。右上角菜单在鼠标移出 2 秒后自动关闭。
- **头像优化**：使用 Next.js `<Image />` 组件优化头像加载，支持 Google 和 Dicebear 远程图片。
- **悬浮控制面板**：游戏内的播放控制和进度条。
- **设备状态指示**：通过颜色和图标直观显示 MIDI 连接状态。
- **响应式设计**：适配不同屏幕尺寸，曲库列表自动调整列数。
- **鲁棒性增强**：完善的国际化语言支持、全局错误处理机制（解决 ChunkLoadError）以及开发环境兼容性优化。
